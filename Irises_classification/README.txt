Примеры классификации растений на основе 'iris dataset' библиотеки scikit-learn.

Модели на основе scikit-learn:
- Поиск оптимальной величины k при использовании модели на основе алгоритма K Nearest Neighbors.
- Поиск оптимальной величины тестовой выборки при использовании модели на основе Logistic Regression.
- Использование инструмента кросс-валидации для выбора оптимальной модели.
- Использование инструмента GridSearchCV для настройки параметров модели ('n_estimators' для RandomForestClassifier).

Модель на основе TensorFlow:
- Выбор параметров, коррелирующих с итоговым значением (меткой класса).
- Использование инструмента регуляризации весов в модели с несколькими слоями.
- Использование инструмента callbacks.EarlyStopping для остановки процесса обучения модели при снижении точности.
