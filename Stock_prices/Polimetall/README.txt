Модель LSTM предсказывает котировки акций "Полиметалла" по данным за предшествующие периоды.

В тестовую выборку входят 90 последних значений котировок, в обучающую выборку - предшествующие данные за весь доступный период наблюдений.

В файле 'Polimetall_stock_price_LSTM.py' - исходный код для загрузки данных и формирования модели.
В процессе обучения используется инструмент EarlyStopping(). Обучение прерывается, если нет улучшения точности в течение 3 эпох.
Выводится график с фактическими и прогнозируемыми значениями для тестовой выборки котировок.

В файл 'Polimetall_model.h5' сохранена модель, показавшая точность прогноза на тестовых данных более 97%.

Источник данных: ИК "Финам" (https://www.finam.ru/profile/moex-akcii/polymetal-international-plc/export)
